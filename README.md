# 🚀 Interactive React Portfolio (Dashboard System)

An interactive portfolio web application built with React, featuring a simulated dashboard system to demonstrate real-world frontend architecture, UI state management, and SPA navigation.

👉 https://portfolio-sanrutai-tangpriyarak.vercel.app/

---

## 🧠 Overview

This project is designed as a **portfolio + dashboard hybrid system**, showcasing frontend engineering skills through interactive UI components and mode switching between portfolio and dashboard views.

---

## ⚙️ Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Framer Motion
- Vercel (Deployment)
- Git & GitHub

---

## ✨ Features

- Portfolio ↔ Dashboard mode switching (SPA architecture)
- Interactive UI with state management
- Simulated dashboard for system visualization (IoT/control system concept)
- Responsive design for all devices
- Component-based scalable structure
